#  Sui Student NFTs

A complete decentralized application (dApp) for minting student-related NFTs on the Sui blockchain.

##  Features

- **Digital Student ID Card** - Official on-chain identity
- **Event Pass NFT** - Access passes for college events  
- **Attendance Report NFT** - Academic performance records
- **Bunk Pass NFT** - Humorous skip-class certificates
- **Hostel Complaint NFT** - Lodge grievances on-chain

### Key Highlights

-  All data stored permanently on Sui Testnet
-  Custom templates for each NFT type
-  Downloadable cards with transaction hash
-  No server-side storage - fully decentralized
-  Premium glassmorphism UI design
